# 聊天质量反馈

## 触发条件

用户反馈 AI 角色回复质量问题，例如：

- 回复重复
- 主体错乱
- 记忆丢失
- 人设偏移
- 回复截断
- 爆 prompt / 爆思维链
- 输出代码、JSON、系统信息
- 图片里展示异常回复

## 目标

这个 Skill 不以立即修复模型为目标，而是完成：

1. 识别问题类型
2. 收集必要证据
3. 判断优先级
4. 提交聊天质量反馈工单
5. 给用户一个有温度的受理回复
